# LessPass cordova

LessPass Android application

## Requirements

 * Cordova (`npm install -g cordova`)

## Check everything ok

    cordova requirements

## Test

    cordova run android
    
## Build Release

    cordova build --release -- --keystore=~/Android/lesspass.keystore --storePassword=... --alias=lesspass --password=...
