# LessPass pure

LessPass web component

 * vue
 * vue-router
 * vuex
 * ava
 * webpack

## Tests

    npm install
    npm test

## Dev

    npm run dev
    
Then open [http://0.0.0.0:8080/](http://0.0.0.0:8080/).

## Build

    npm run build
